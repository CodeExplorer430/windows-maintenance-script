# Security Policy

## Supported Versions

We recommend using the latest version of the Windows Maintenance Framework to receive security updates.

| Version | Supported          |
| ------- | ------------------ |
| 4.1.x   | :white_check_mark: |
| < 4.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do not open a public issue. Instead, please report it via [INSERT METHOD, e.g., Email].

When reporting, please include:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact of the vulnerability.

We strive to acknowledge all security reports within 48 hours and provide a fix or mitigation plan shortly after.

Thank you for helping keep this project secure!
